게으른 달걀 (Lazy Egg Trading)

채팅형 주식 트레이딩 플랫폼 사업계획서

1. 사업명

게으른 달걀 (Lazy Egg Trading)
이름 유래: "게으른" 투자자를 위한 쉬운 플랫폼, "달걀"은 재테크의 기본처럼 안정적이고 성장하는 이미지를 상징. 영문명으로 글로벌 확장 용이.

2. 형태

모바일 앱 기반 채팅형 플랫폼 (PWA 지원)

  • iOS/Android 앱으로 출시, 웹 브라우저에서도 접근 가능 (Progressive Web App)
  • 백엔드: FastAPI (Python)
  • 프론트엔드: React 18 + Tailwind CSS + WebSocket(텔레그램)으로 실시간 채팅 구현
  • API 연동: 오픈뱅킹 (금융결제원) + 증권사 Open API (키움증권 중심)
  • 배포 플랫폼: Google Cloud Run (Serverless 컨테이너)
    → 이유: 코드 푸시 즉시 배포, 자동 스케일링, 비용 최적화 (사용량만큼만 과금, 무료 티어 대폭 활용)
  • 도메인: 커스텀 도메인 연결 (예: lazyegg.kr) → Google Cloud에서 무료 SSL 자동 발급

3. 타겟

주요 타겟: 10~30대 초보 투자자 (대학생, 신입 직장인, 주식에 관심 있지만 복잡함에 포기한 층)

  • 부타겟: 40대 이상 바쁜 직장인 (시간 부족으로 간단한 투자 원함)
  • 사용자 규모 목표: 출시 1년 내 10만 명 다운로드 (초보자 중심 마케팅)

4. 문제점 → WOW 포인트로 전환

우리는 고객의 페인포인트를 WOW 포인트로 바꿉니다!

기존 주식 투자에서 가장 큰 페인포인트는 복잡한 용어와 인터페이스, 메뉴 링크조차 제대로 안 되는 채팅앱, 감정 언어도 이해하지 못하고 상담원 연결만 반복하는 무늬만 채팅앱, 불완전한 이해로 인한 불완전 판매 위험입니다.

게으른 달걀은 이를 전문 금융 통역 AI Agent로 완벽하게 대응합니다.
고객이 아무렇게나 자연어로 말해도 RAG(Retrieval-Augmented Generation)를 활용해 표준화된 정확한 한국어 스크립트로 변환하고, 전문 통역사 수준의 명확하고 친절한 설명을 제공합니다.

결과:
• 응대 시간이 획기적으로 단축
• 불완전 판매를 원천적으로 예방
• 실제 전문 통역사와의 블라인드 테스트에서 90점 이상 압승을 기록

복잡하고 무서운 주식 투자를 → 쉽고 재미있고 안전한 대화로 바꾸는 것,
이것이 게으른 달걀이 만드는 진정한 WOW 포인트입니다.

5. 핵심솔루션 (기능 상세)

핵심 기술: 전문 금융 통역 AI Agent (RAG 기반 자연어 → 표준 스크립트 변환)

주요 기능

  1. 오픈뱅킹 기반 회원가입/조회: 은행/증권 계좌 연동으로 간편 가입. 거래내역 통합 조회
  2. 모의주식 모드: 무제한 가상 투자 연습 (키움 Open API 모의투자 연동, 백터DB)
  3. 채팅형 실전 거래: “삼성전자 10주 사줘” 입력 → AI가 전문 통역사처럼 확인·설명 후 키움 API로 주문 실행
  4. AI 추천/감정 케어: “오늘 추천주?” → 정확한 근거 설명. “우울해” → 위로 + 리밸런싱 제안
  5. 탈퇴 자동 청산: 거래내역 확인 → 보유 주식 자동 매도 → 잔고 이체 → 앱 탈퇴 (증권계좌 폐쇄는 사용자 안내)

기술 상세: RAG 기반 금융 전문 통역 AI, MongoDB(포트폴리오), Elasticsearch(종목 검색), Gemini/Claude API(자연어 처리)
배포 아키텍처: GitHub → Cloud Build/Artifact Registry → Cloud Run 자동 배포
CI/CD: GitHub Actions workflow (Workload Identity Federation으로 키리스 인증 – 최고 수준 보안)

6. 가치

  • 사용자 가치: 불완전 판매 걱정 없이 안전하고 쉬운 투자 경험
  • 사회적 가치: 금융 문맹률 감소, 불완전 판매 예방으로 건전한 투자 문화 조성
  • 경제적 가치: 응대 효율화로 비용 절감, 개인 자산 증대

7. 시장규모

  • 글로벌 핀테크 시장: 2025년 약 395억 달러, CAGR 16.2%로 2032년 1,127억 달러 예상
  • 한국 핀테크 시장: 디지털 결제 부문 중심 빠른 성장세, 주식 트레이딩 앱 시장 1~2억 달러 규모 추정
  • 타겟 시장 기회: 한국 활성 투자자 1,000만 명 이상, 핀테크 투자 활발

8. 수익모델

기존 증권사의 '무늬만 채팅앱'을 우리의 진짜 채팅형 앱으로 대체

  • 대부분 증권사 앱의 채팅 기능은 메뉴 링크조차 제대로 연결되지 않고, 고객의 감정 언어도 이해하지 못한 채 “상담원 연결”만 반복합니다.
  • 게으른 달걀은 이 한계를 넘어 실제 거래까지 가능한 진정한 채팅형 플랫폼으로 대응합니다.
  • 초기 운영 비용 최소화: Google Cloud Run 무료 티어 + GitHub Actions 무료 분량 활용 → 출시 초기 월 비용 거의 0원
  • 성장 시: 트래픽 증가해도 사용량 기반 과금으로 효율적 관리
  1. 거래 수수료 제휴: 실전 거래 시 증권사 수수료 일부 공유 (기존 앱의 불편한 채팅 대신 우리 앱으로 유도 → 거래량 증가)
  2. 프리미엄 구독: 월 4,900원 – 고급 AI 분석, 실시간 알림, 세금 계산 도구
  3. 모의주식 대회 스폰서십: 기업 광고/상금 유치
  4. 교육 콘텐츠 판매: AI 생성 초보자 강의 패키지

예상 수익: 출시 1년 내 매출 10억 원 목표 (구독 50%, 제휴 거래 수수료 30%)

9. 차별성

  • 진짜 작동하는 채팅형 거래: 기존 증권사 '무늬만 채팅' vs. 우리의 실제 거래·감정 이해 가능한 AI Agent
  • 전문 금융 통역 AI Agent: RAG로 자연어 → 정확한 금융 스크립트 변환, 불완전 판매 예방
  • 감정 케어 AI: 투자 스트레스 완화
  • 모의+실전 seamless 전환: 나이 제한 없이 초보자 유입 강점
